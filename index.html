About the Blockz Handheld Game Device

The Blockz hand held Device is a portability focused handheld gaming device pre installed with the game Blockz and the device has many features such as:

  • Small portable design for on the go Gaming
  • Good battery life up to 20 hours per battery change
  • 25 individual LED display which can be seen in sunlight
  • USB interface so the games console can be used without batteries

About the Game

Blockz is a puzzle game where you have to fill in all 5 LED horizontally and this is made harder by there being 4 diffrent brick veriants and the spawn of these Brickz are random decided by the console, But when you have created a Horizontel line it will count as 1 point although if you fill a line vertically there will be no more room for more bricks to be laid and this will result in a game end.


How to play the game?

The following Inputs are how to play the Brickz Game device.

  • Power on the device and get ready to start playing!
  • A button: Move the current brick to the left
  • B button: Move the current brick to the right
  • Pressing both A and B buttons will rotate the current brick Clockwise



Design of the device casing

The desing of the Blocks Handheld Gaming device is oriented around the portability if the device so this means we had to design the device to be small enough to fit in someones hand and also small enough to fit inside of a pocket whilst not in use.



Packaging

The packaging for our device is the first thing which the user will see so we have had countless hours poured into developing packaging which not only fits the design criteria for the Blockz Handheld system but to also capture the simplicity of the device on the front. We have kept our design simple so you can focus on whats inside the box and not whats on the outside.
